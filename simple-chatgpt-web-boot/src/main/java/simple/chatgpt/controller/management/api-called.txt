userList.js API Calls

Action	HTTP Method	Endpoint	Description
Load user lists (paged/search)	GET	${API_USERLIST}?<query>	Fetches paginated user lists based on search params.
Load by ID	GET	${API_USERLIST}/get?listId=<id>	Fetch a single user list by ID.
Delete user list	DELETE	${API_USERLIST}/delete?listId=<id>	Deletes a user list by ID.

userListMember.js API Calls

Action	HTTP Method	Endpoint	Description
Load members (paged/search)	GET	${API_USERLIST_MEMBER}/search?<query>	Fetch paginated members for a list.
Load member by ID	GET	${API_USERLIST_MEMBER}/get?memberId=<id>	Fetch a single member by ID.
Save member (create)	POST	${API_USERLIST_MEMBER}/create	Create a new member in a list.
Save member (update)	PUT	${API_USERLIST_MEMBER}/update?memberId=<id>	Update an existing member.
Delete member	DELETE	${API_USERLIST_MEMBER}/delete?memberId=<id>	Delete a member by ID.
